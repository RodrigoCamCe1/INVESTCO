Diagramas de Casos de Uso — Investco

Este documento contiene los 41 diagramas detallados de casos de uso del sistema Investco, generados siguiendo la convención UML estándar usada en el documento original del proyecto. Cada diagrama incluye los actores principales (figuras stick), el caso de uso central (óvalo), las relaciones <<include>> hacia validaciones y cálculos del sistema (arriba-derecha) y las relaciones <<extend>> hacia acciones alternativas o de cierre (abajo). Cada página está pensada para imprimirse y recortarse individualmente para reemplazar los diagramas correspondientes en el documento original.
